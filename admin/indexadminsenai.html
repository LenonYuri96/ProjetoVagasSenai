Como usar o painel de administração

Este painel permite gerenciar as mídias exibidas no sistema de vagas do SENAI.

Você pode fazer upload de arquivos de duas maneiras:

  1. Arrastar e soltar os arquivos na área designada
  2. Clicar no botão "Selecionar Arquivos" para escolher manualmente

Formatos suportados: JPG, PNG, GIF, MP4, WEBM

Tamanho máximo: 30MB por arquivo

Para cada item carregado, você pode:

  • Definir o tempo de exibição (para imagens)
  • Excluir itens que não são mais necessários
  • Visualizar como o item aparecerá no sistema
